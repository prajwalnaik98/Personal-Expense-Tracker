\# Personal Expense Tracker



A Python application to record, manage, and summarize personal expenses using CSV file handling.



\## Features



\- Add expense records

\- Store expenses in a CSV file

\- Display all expenses

\- Show category-wise spending summary

\- Find the highest expense



\## Technologies Used



\- Python

\- CSV

\- File Handling

\- datetime

\- os



\## How to Run



1\. Make sure Python is installed.

2\. Run the following command:



```bash

python expense\_tracker.py

```



The program will automatically create an `expenses.csv` file if it does not exist.
